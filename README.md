<!-- GETTING STARTED -->
## Getting Started

This is an example of how you can you set up SSO project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

You will need to make sure your server meets the following requirements:
1. PHP >= 7.1.3
2. Composer >= 7.1.3
3. MySQL >= 5.6

### Installation

1. Get a free API Key at [https://github.com/28thsly/api.single-sign-on/issues](https://github.com/28thsly/api.single-sign-on/issues)
2. Clone the repo
   ```sh
   git clone https://github.com/28thsly/api.single-sign-on
   ```
3. Install packages
   ```sh
   composer install
   ```
4. Run the App in `terminal`
   ```js
   php artisan serve
   ```
